How the internet works

Nov. 13th, 2014

When your computer is connected to the internet, you are connecting to an Internet Service Provider (ISP). The ISP server provides the link between your browser and the rest of the internet. The server recieves requests from the browser(s) to view the webpage(s).Your ISP server looks in a huge database of internet addresses to find the host server which holds the website in question, then sends the host server a request for the page. Host server sends the requested page to the ISP server. ISP sends the page to the browser and you see it displayed on your screen.
